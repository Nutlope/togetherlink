Get started

1
Install with the one-liner above. It drops the binary at ~/.togetherlink/bin/ and adds togetherlink, tclaude and topencode.
2
Run topencode (or tclaude). On first launch it asks once for your Together API key — press Enter to skip and add it later.
3
That's it. Your tool runs against Together models and stays up to date on its own.
