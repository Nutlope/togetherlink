OpenCode & Claude Code

One small, self-updating binary. Install in a second, run topencode or tclaude, and your existing tools route through Together AI models like GLM 5.2 — no proxy to run, no config to write.

$ curl -fsSL https://togetherlink.vercel.app/install.sh | sh
Works on macOS & Linux · installs Bun for you if needed · auto-updates in the background
Supported

OpenCode

Run topencode and OpenCode launches with Together GLM 5.2 wired in — config injected, nothing written to disk.

Supported · optimizing

Claude Code

Run tclaude and Claude Code routes through a local translation proxy. A lighter shared-daemon build is coming via auto-update.

Auto-updating

Stays current

The binary checks for new versions in the background and swaps itself atomically. Your topencode command never changes; it just gets better.

Get started

1
Install with the one-liner above. It drops the binary at ~/.togetherlink/bin/ and adds the togetherlink, tclaude and topencode commands.
2
Run topencode (or tclaude). On first launch it asks once for your Together API key — press Enter to skip and add it later.
3
That's it. Your tool runs against Together models. Updates arrive automatically; opt out anytime with TOGETHERLINK_NO_AUTOPDATE=1.
